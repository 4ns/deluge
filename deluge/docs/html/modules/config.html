:mod:’deluge.config’¶

Configuration class used to access/create/modify configuration files.

class deluge.config.Config(filename, defaults=None, config_dir=None)¶

This class is used to access configuration files.

apply_all()¶
Runs all set functions
get(key)¶
Get the value of ‘key’. If it is an invalid key then get() will return None.
get_config()¶
Returns the entire configuration as a dictionary.
get_previous_config()¶
Returns the config prior to the last set()
load(filename=None)¶
Load a config file either by ‘filename’ or the filename set during construction of this object.
register_change_callback(callback)¶
Registers a callback that will be called when a value is changed
register_set_function(key, function, apply_now=True)¶
Register a function to be run when a config value changes.
save(filename=None)¶
Save configuration to either ‘filename’ or the filename set during construction of this object.
set(key, value)¶
Set the ‘key’ with ‘value’.
